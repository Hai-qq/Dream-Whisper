# 梦语 (Dream Whisper) - 项目说明文档

## 1. 项目简介 (Project Overview)

**“梦语” (Dream Whisper)** 是一款融合心理学理论与生成式 AI 技术的梦境分析与可视化应用。

### 核心问题解决
人们常对自己的梦境感到困惑，渴望了解其中隐藏的潜意识信息，但缺乏专业的心理学知识解读。同时，梦境的视觉画面往往转瞬即逝，难以向他人精准描述或留存。

**本项目致力于解决：**
1.  **解梦门槛高**：用 AI 替代昂贵的心理咨询师，提供即时、专业的梦境解析。
2.  **画面留存难**：将抽象的文字描述转化为具象的高质量图像和视频，让梦境“看得见”。
3.  **自我认知盲区**：通过梦境数据量化分析用户的潜意识人格特征。

---

## 2. 业务价值 (Business Value)

### 对用户的价值
*   **情感疗愈与自我发现**：通过深度解析（荣格/弗洛伊德流派），帮助用户理解焦虑、恐惧或渴望的根源，缓解心理压力。
*   **独特的娱乐体验**：将梦境变成“微电影”，满足猎奇心理和分享欲（社交货币）。
*   **数字化记忆资产**：建立个人的“梦境日记”，长期记录可追踪心理状态变化。

### 商业与社会价值
*   **心理健康普惠**：作为轻量级的心理干预工具，降低大众接触心理健康服务的门槛。
*   **IP 孵化潜力**：用户生成的大量梦境故事和视觉素材，可作为小说、游戏或影视的创意灵感库。

---

## 3. AI 创新性说明 (AI Innovation)

AI 是本项目的核心驱动力，而非简单的锦上添花。

### 核心 AI 应用点
1.  **深度心理分析 Agent (GLM-4)**：
    *   **创新点**：并非简单的关键词匹配，而是通过 Prompt Engineering 构建了具备“采访者”人格的 AI。它能主动进行多轮对话，引导用户回忆感官细节（颜色、声音、情绪），并模拟专业咨询师进行流派化的深度解读。
2.  **多模态视觉重构 (Volcengine Ark / Doubao)**：
    *   **创新点**：实现了“文本 -> 增强 Prompt -> 图像 -> 视频”的自动化链路。系统会自动提取梦境中的关键视觉元素，重写为适合绘画模型的提示词，生成的画面风格统一（唯美动漫风），高度还原梦境氛围。
3.  **潜意识数据量化**：
    *   **创新点**：利用 LLM 将非结构化的梦境文本转化为结构化的五维人格数据（创造力、逻辑性、情感度、灵性、现实感），并生成动态雷达图。

## 4. 技术实现说明 (Technical Overview)

项目采用现代化的全栈 Web 架构，确保高性能与良好的扩展性。

### 架构设计
*   **前端框架**：Next.js 15 (React) - 提供流畅的 SPA 单页体验和服务器端渲染 (SSR) 的 SEO 优势。
*   **后端服务**：Next.js API Routes (Serverless) - 处理业务逻辑、API 转发与密钥安全管理。
*   **AI 模型层**：
    *   **语义理解**：智谱 AI (GLM-4) - 负责对话引导与心理分析。
    *   **视觉生成**：火山引擎方舟 (ByteDance Doubao模型) - 负责 Text-to-Image 和 Image-to-Video 生成。
*   **部署架构**：Docker 容器化部署 - 基于 Node.js 20-alpine 镜像，支持一键部署至 ModelScope 等云平台。

### 关键流程
1.  **用户输入** -> **AI 采访 Agent** (多轮对话补充细节)。
2.  **提交分析** -> **LLM 结构化输出** (JSON 包含分析报告、人格评分、绘图 Prompt)。
3.  **绘图 Prompt** -> **图像生成 API** -> **返回图片**。
4.  **图片 + 动态指令** -> **视频生成 API** -> **返回视频流**。

## 5. 交互与设计说明 (Interaction & Design)

### 设计理念
*   **沉浸式 (Immersive)**：采用深色星空动态背景，营造静谧、神秘的梦境氛围，引导用户放松潜意识。
*   **对话式引导 (Conversational)**：摒弃枯燥的表单填写，通过类似聊天的界面引导用户一步步回忆梦境。

### 关键界面流程
1.  **梦境记录页**：
    *   简洁的聊天窗口，AI 根据用户的只言片语主动追问（如“当时的光线是怎么样的？”）。
    *   支持语音输入（规划中），降低记录门槛。
2.  **分析结果页 (The Reveal)**：
    *   **视觉冲击**：首屏展示 AI 生成的梦境视频，自动循环播放，瞬间以此抓住用户注意力。
    *   **深度解读**：下滑查看卡片式的心理分析报告，重点突出“象征意义”和“生活启示”。
    *   **人格雷达**：动态绘制的五维图表，直观展示本次梦境反映的心理状态。
